Hi,

is there any way to convert a dask DataFrame back to Pandas? I have some features I need, which aren't yet implemented in Dask. However I need parallel / partitioned mapping.

import dask.dataframe as dd
my_dask_ df = dd.from_pandas(df, npartitions=4)
my_dask_df.map_partitions(...)
del df

What I am looking for is:

df = my_dask_df.to_pandas()
del my_dask_df

Best,
Marius

You can use get_partition to compute and fetch parts of the dataframe. This isn't streaming, whole tasks get performed, that is how Dask works. You could conceivably use streaming dataframes from the Streamz project, but I'm pretty sure that's not what you are after.
